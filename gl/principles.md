  irreversible legal/financial signatures,

## Build Doctrine
- Phase/date rule: frame by phase+function, never calendar
- Hard external dates only: Jun 1, Sep 30, Aug 1
- Opus: novel reasoning only. Sonnet: all execution.
- Extended thinking: Opus only, never Sonnet

- Every CC/Cursor prompt ends with git commit + push
- Preflight.ps1 before every push

  Jason explicitly says "simulate."

## Response Doctrine

- ALL TESTING IS REAL — never simulate

## Thread Naming (canonical)
Format: [GL | WORKSTREAM | Topic · Topic | YYMMDD]
- Leading namespace always "GL" — never project name
- Separator between topics: middle-dot "·" (U+00B7)
- Date = START date of chat, never current date
- If uncertain of start date: ask Jason

  "empower," "let's get started," "circle back," "tracks,"
